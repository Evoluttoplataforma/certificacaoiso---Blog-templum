# Saiba como identificar aspectos e impactos ambientais

Primeiramente, para uma organização atender aos requisitos de gestão ambiental estabelecidos pela norma [ISO 14001](https://certificacaoiso.com.br/iso-14001/), de 2015, é necessário compreender os significados dos termos aspectos e impactos ambientais. Um dos requisitos é estabelecer, implementar e manter procedimentos para identificar os aspectos e impactos ambientais decorrentes das atividades de uma organização.

**Mas qual a diferença entre eles?**

Aspectos ambientais são entendidos como elementos das atividades, produtos ou serviços de uma organização. Eles podem interagir com o meio ambiente, causando ou podendo causar impactos ambientais, positivos ou negativos.

> Saiba mais sobre a [ISO 14001.](https://certificacaoiso.com.br/iso-14001/)

Impactos ambientais são quaisquer modificações do meio ambiente, positiva ou negativa, resultante ou não dos aspectos ambientais da organização.

Uma vez que você entende a diferença conceitual entre eles, é preciso descobrir como se avalia isso na prática. Afinal, **como descobrir se é um aspecto ou um impacto ambiental**?

A melhor forma de fazer isso é através de uma metodologia comparativa, em colunas. Isso porque, como **os aspectos ambientais causam os impactos**, só é possível compreender a natureza de um em relação ao outro.

Resumindo, aspecto ambiental é a causa e impacto ambiental é o efeito. Veja o exemplo abaixo:

A identificação de uma organização é umas das etapas mais importantes da implementação do [sistema de gestão ambiental](https://certificacaoiso.com.br/iso-14001-gestao-ambiental-que-abre-portas-para-novos-negocios/).

Uma das maneiras de realizar esta identificação pode ser a construção de uma matriz que relacione os aspectos e impactos ambientais. Esta ferramenta auxilia também na classificação e determinação de sua significância.

É muito importante para uma empresa **identificar os riscos ambientais de uma operação** com antecedência, para então agir para minimizá-los o máximo que for possível. Dessa maneira, você pode, por consequência, diminuir os impactos [ambientais](https://certificacaoiso.com.br/indicadores-ambientais-sustentabilidade/) de forma significativa.
